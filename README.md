## Description

A nestjs application that has endpoint which get ethereum addresses and check their balances.

## Environment Variables
Fill the .env file for variables.

PORT=
MAINNET_KEY=
COINGECKO_URL=

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
